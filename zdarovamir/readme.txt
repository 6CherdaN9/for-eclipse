Вводить данные в файл "input.txt" нужно в следующем порядке:
 (тип данных) (операция) (данные)
 Типы данных: s - числа, v - вектора
 Операции: +, -, *, /, ^, ! - для чисел; +, -, * - для векторов
 Данные: для числа данные состоят из чисел, с которыми будет проводиться операция
	 для векторов данные состоят из количества координат у вектора и координат двух векторов
 Пример для векторов:
	v + 2 3 5 4 9 - сложение векторов (3, 5) и (4, 9) 
	v * 3 4 8 2 0.6 4 7 - скалярное произведение векторов (4, 8, 2) и (0.6, 4, 7)
 Пример для чисел:
	s * 6 3 - умножение шести на три
	s ! 4 - факториал четырех(для вычисления факториала вводится одно целое число)
